# 开始

ReactNative端基座，所有需要支持app的模块或项目都需要依赖此模块

## 安装

```bash
yarn duxapp app add duxappReactNative
```

## 教程

如果你还不知道如何使用RN端，请先阅读 [RN端教程](/docs/course/rn/start)

这个模块中还提供了 用户协议、版本更新、权限管理等基础功能
