## Props

继承自Taro的[Switch Props](https://nervjs.github.io/taro-docs/docs/components/forms/switch)

### defaultValue

默认值

| 类型 | 必填 | 默认值 |
| ---- | -------- | ------- |
| boolean \| any | 否 |  |

### values

指定未选中和选中的值

| 类型 | 必填 | 默认值 |
| ---- | -------- | ------- |
| [any, any] | 否 |  |
