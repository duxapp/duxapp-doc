# 2026-01-05 TabBar 悬浮模式 其他常规更新

## 基础库
- loading 修复在宽屏上不居中的问题 
- TopView 新增 pageMetaProps 属性，可以指定小程序端 page-meta 组件属性
- 新增 usePageShow useWindowInfo hook
- route.back() 方法如果无法返回（已经是一一个页面），会自动返回主页
- 完善Types

## UI库

- showContextMenu 每一项新增 props (TextProps) 参数 可以指定这个菜单的属性
- Radio 新增 multiple 属性，开启后将变成多选
- 修复 List 小程序端虚拟列表多个问题
- List 组件 新增 reloadType 如果指定为 top 只有列表在顶部的情况下才会重新刷新数据
- TabBar 新增 floating 开启后，可以让TabBar悬浮在底部
- TabBar 新增 position 属性，可以让菜单展示在左侧
- TabBar 新增 defaultIndex 属性，用于指定默认显示第一个TabBar
- TabBar 新增 setVisible() 方法 可以设 TabBar的显示和隐藏

## React Native

- 修复一些样式不生效
